Abra o projeto em um ambiente de desenvolvimento Python, como o PyCharm. Em seguida, abra o terminal do projeto e execute o comando:

```bash
python manage.py runserver
```

Depois que o servidor iniciar corretamente, abra o navegador e acesse o endereço informado no terminal para entrar no painel administrativo. Na maioria dos casos o link será:

- http://127.0.0.1:8000/admin/
- http://127.0.0.1:8000/faq/

O endereço pode ser diferente no seu computador, portanto utilize o que aparecer no terminal.
